#### To set up and run the project locally.

```bash
git clone https://github.com/jaydhumal23/lAuth.git
cd lAuth
```

#### Install Dependencies

```bash
npm install
```

#### Create .env file and setup

```bash
mongourl="add db url"
PORT=3000
jwtpass="createjwtpass"
```

#### Run the App

```bash
npm run app
```

``
server running at localhost:3000
``
